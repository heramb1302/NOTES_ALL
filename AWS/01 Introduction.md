
## AWS - Introduction Summary

Definition: AWS (Amazon Web Services) is a comprehensive cloud computing platform owned by Amazon.

Service Model: It provides IT services (computing power, storage, databases, etc.) delivered over the internet.

Pricing: Operates on an on-demand, pay-as-you-go model (you only pay for what you use).

Scale: * Launched in 2006.

Offers a massive catalog of 200+ services.


## **AWS Market Position & Customers**

- **Market Leader:** AWS holds the **largest share** of the global cloud market (approx. **30%**), making it the top industry leader.
    
- **Diverse User Base:** Services are used by everyone from small startups to massive global enterprises and government agencies.
    
- **Major Clients:** Key notable users include:
    
    - **Entertainment/Tech:** Netflix, Samsung, Airbnb.
        
    - **Government/Science:** NASA and various worldwide government bodies.


## **AWS - Core Service Domains**

- **Definition:** Core cloud service domains provide the **foundational IT resources** necessary to build, run, and power applications in the cloud.
    
- **Primary Domains (The Building Blocks):**
    
    - **Compute:** (Highlighted in the slide) Provides processing power and virtual servers to run applications.
        
    - **Storage:** Provides secure space for saving and retrieving digital files and data.
        
    - **Networking:** Connects services together and manages traffic flow over the internet.
        
- **Supporting Services:** These work alongside the primary domains to create complete infrastructures:
    
    - Databases
        
    - Security
        
    - Management Tools
